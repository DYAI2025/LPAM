## Problem Statement

| Field | Value | Source Type | Source |
|---|---|---|---|
| Problem Statement | A working personal-AI stack (Nous Hermes agent + gbrain long-term memory + model fallback + desktop-app-to-VPS + local voice) existed only as bespoke, un-transferable infra on one operator's machines (hardcoded `/root` paths, inlined secrets, no `.env.example`, an `install.sh`↔`.gitmodules` contradiction, no frontend clone). It must become a single git-pull-deployable kit a stranger can stand up from the shipped docs alone, without redistributing third-party software and without leaking secrets. | EXPLICIT | SRC-001, SRC-005 |
